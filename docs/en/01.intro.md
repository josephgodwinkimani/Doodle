## Server Requirements

- PHP >= 7.0
- PDO PHP Extension
- cURL PHP Extension
- FTP Extension
- Mbstring PHP Extension
- Fileinfo PHP Extension
- Zip PHP Extension
- Composer

## Development Environment

- Git bash or Unix Shell/Terminal
- [php-docker][1] _(recommended)_
